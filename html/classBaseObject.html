Detailed Description

Base multimedia-type file object.

This class is the base class for all objects which describe multimedia files in the project.

void BaseObject::print ( ) const

Prints formatted description of object.

This function prints out a description of the object, usually by displaying the values of each member variable. It does this by calling the toString() function of the object. Thus that function is virtual, so that subclasses of BaseObject can effectively reimplement this function to include extra member data.
