\subsection{Detailed Description}
Base multimedia-\/type file object. 

This class is the base class for all objects which describe multimedia files in the project.

This function prints out a description of the object, usually by displaying the values of each member variable. It does this by calling the \hyperlink{classBaseObject_af433968d291ebfc54c701987047f9348}{to\-String()} function of the object.
